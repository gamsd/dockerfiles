Beautiful collaborative presentations

with Reveal.JS and Docker

Build beautiful presentations with reveal.js and zero configuration effort!
This is a sample presentation based on reveal.js and using the [gamsd/revealjs](https://hub.docker.com/r/gamsd/revealjs/) docker image
I wrote about this on my blog [Check it out!](http://blog.gamsd.com/revealjs-docker-presentations)
Inspired by Peter [Parente](https://github.com/parente)'s [work](https://mindtrove.info/a-reveal.js-docker-base-image-with-onbuild/) Customized for [my](https://github.com/gamsd) own [tastes](https://github.com/gamsd/dockerfiles)
